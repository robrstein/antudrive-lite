# AntuDrive Lite — Led Light App Control

> Versión gratuita y de código abierto del proyecto **AntuDrive**, aplicación Flutter para control de iluminación automotriz vía **Bluetooth Low Energy (BLE)**.

## 📋 Descripción
Esta versión **Lite** incluye las funciones esenciales para controlar luces LED CCT (blanco cálido/frío) en vehículos. El código fuente se publica con fines educativos y experimentales, sin incluir las características premium de la versión **AntuDrive Pro**.

## ⚙️ Funcionalidades
- Control básico de temperatura de color (3000K–8000K)
- Ajuste de brillo (5%–100%)
- Escaneo BLE y conexión a dispositivos compatibles
- Persistencia local de ajustes

> Algunas funciones avanzadas (grupos, temas, presets, sincronización multi-dispositivo) están disponibles solo en **AntuDrive Pro**.
